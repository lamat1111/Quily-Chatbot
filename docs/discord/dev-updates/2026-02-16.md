## LaMat — 11:07 AM UTC

**Updates for Quorum Development**
January 10 – February 16, 2026

This list is for the web app, these changes are not yet live but you can test them on https://test.quorummessenger.com/

🎉 New Features

🛡️ Encrypted DM Backup & Restore
• Export all your DM history as an encrypted backup file (.qmbak)
• Import backups on any device to restore your conversations securely
• Found in Settings > Privacy with clear progress feedback

⚙️ Danger Zone — Reset App Data
• New "Danger Zone" section in User Settings for a full app reset
• Type-to-confirm safety step prevents accidental data loss
• Clears all local data so you can start fresh if needed

🏷️ Smarter Mention Dropdown
• The @mention and #channel dropdown now appears instantly as you type
• Shows your most relevant contacts first, then narrows as you keep typing
• Cleaner, more compact look with better visual selection

😄 Reaction Tooltips with Names
• Hover over any reaction to see who reacted (up to 3 names shown)
• Click "+X more" to open a full modal with all users and their avatars

♿ Accessibility Overhaul (WCAG 2.1 AA)
• Full keyboard navigation with visible focus indicators across the app
• Modals now trap focus properly and restore it when closed
• Screen reader support with ARIA labels, landmarks, and live announcements
• All buttons are now real HTML buttons with proper disabled states
