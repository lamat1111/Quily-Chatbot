# dev-updates - March 30, 2026

## LaMat — 2:57 PM UTC

**New public repo: q-stream-recap**

We've open-sourced the pipeline we use to create the Quilibrium live stream recaps.

It automates the full workflow: download streams from any platform (Twitch, YouTube, X), transcribe them using open-source Whisper models via Groq, generate formatted recaps, and upload to the Quilibrium YouTube channel.

**What's inside:**
⦿ Python scripts for download, transcription, and YouTube upload
⦿ Workflow prompts that work with Claude Code, OpenCode, or any AI coding agent
⦿ Quilibrium glossary that auto-corrects common transcription errors (Klearu, Neynar, etc.)
⦿ Transcriptions and recaps will accumulate over time as we process more streams

**Want to contribute?**
Anyone can run the pipeline with just a free Groq API key. No YouTube channel access needed. You can help by adding translations of existing recaps, improving the scripts, or implementing new features.

Setup instructions: https://github.com/Quilibrium-Community/q-stream-recap
