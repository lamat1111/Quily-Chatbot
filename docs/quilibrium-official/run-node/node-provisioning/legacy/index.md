# Legacy Methods

The following legacy methods can be used to provision and run a node.
They are not actively maintained and *may* still be functional.

:::tip
For most users, [QTools](/docs/run-node/node-provisioning/qtools) or the [Manual Install](/docs/run-node/node-provisioning/manual-install) are preferred over the legacy methods below.
:::

| Method | Description |
|--------|-------------|
| [**Release Autorun Script**](/docs/run-node/node-provisioning/legacy/release-autorun) | Automatically downloads, runs, and updates the node binary via a shell script. |
| [**QOne Script**](/docs/run-node/node-provisioning/legacy/qone) | Community-maintained script that simplifies node setup on Linux. |
