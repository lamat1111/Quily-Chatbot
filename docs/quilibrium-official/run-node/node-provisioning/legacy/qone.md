# QOne Script

QOne is a community-maintained installation script that simplifies node setup on Linux.

:::warning
QOne is a third-party community tool and is not officially maintained by Quilibrium.
Always review scripts before executing them.
:::

## Installation

```bash
wget -O - https://raw.githubusercontent.com/lamat1111/QuilibriumScripts/master/qone.sh | bash
```

For more information, see the [QOne GitHub repository](https://github.com/lamat1111/QuilibriumScripts).
